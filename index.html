5. Data Security

5.1 How We Protect Your Data

  • Password Hashing - Passwords are hashed using bcrypt (industry standard)
  • Encryption in Transit - All data transferred using SSL/TLS encryption
  • Secure Servers - Hosted on secure cloud infrastructure (AWS/PostgreSQL)
  • Access Controls - Only authorized personnel access user data
  • Regular Updates - Security patches applied promptly

5.2 Limitations

While we implement industry-standard security measures, no method is 100% secure. You use the App at your own risk, though we work continuously to improve security.
